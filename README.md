# What's next?

```sh
go install github.com/xhd2015/whats_next@latest

whats_next show
```

# Usage
```sh
whats_next
user> ...(your follow-up)
```

Show brief rules:
```sh
whats_next show
```

Add custom rules
```sh
whats_next add --title "Reading Files" "Prefer to use batch_read_file instead of read_file for multiple files reading"
